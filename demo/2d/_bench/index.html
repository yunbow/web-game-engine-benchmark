Bench Harness — 自動FPS計測（13テーマ × 7 JSエンジン）

FPSは各ゲームのiframe内 requestAnimationFrame をハーネスが直接計測（ゲーム無改変）。負荷は合成キーで「+」を送って段階制御。Chrome推奨・このタブを前面のまま放置（背面タブはrAFが絞られ無効）。

サーバ前提

リポジトリの js/ をルートにHTTP配信して、/_bench/ を開くこと。
例) cd js && python -m http.server 8000 → http://localhost:8000/_bench/

テーマ

エンジン

負荷レベル（"+"の押下回数）

各レベルでデフォルトから累積でこの回数だけ "+" を送る。SPECのSTEPは全エンジン共通なので同レベル＝同負荷設定。

タイミング(ms)

待機中。左で対象を選び「計測開始」。
