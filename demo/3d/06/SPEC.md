# 3D テーマ6（T5）共通仕様 ― 動的シャドウ光源（多数のリアルタイム影マップ）

全ライブラリ（three.js / Babylon.js / PlayCanvas / A-Frame）で**同一仕様**を実装し、性能を横並び比較するための共通仕様。
**比較の主軸 = 影を落とすリアルタイム光源の数（シャドウマップ枚数）**。2Dテーマ11（2Dライティング/影）の3D版で、本物のシャドウマップを使う。

> この SPEC.md が唯一の正。柱配置・光源パラメータ・影解像度・カメラ・HUD を4ライブラリで揃える。

## ゲーム内容
- 地面に**柱（ボックス）をグリッド配置**し、上空を**複数のスポットライトが周回**しながら**動的な影**を落とす。
- **目的は性能比較**: `+`/`-` で**影を落とす光源の数**を増減し、シャドウマップ生成パスのコストで FPS を測る。

## 描画バックエンド
- **全ライブラリ WebGL2 で統一**。各エンジンのリアルタイム**シャドウマップ**（three=ShadowMap / Babylon=ShadowGenerator / PlayCanvas=light.castShadows / A-Frame=three ShadowMap）を使う。
- シャドウマップ解像度は **1024 x 1024**（4ライブラリ共通）。フィルタは各エンジンの PCF 系（ソフト影）。

## 画面・カメラ
- 描画サイズ **960 x 540（16:9 固定）**。背景 `#0a0c12`。**Y軸上向き**。地面 y=0。
- カメラ固定: 位置 **(0, 28, 40)** / 注視 **(0, 2, 0)** / fov 55° / near 0.5 / far 500。
- 弱い環境光（影が真っ黒に潰れない程度。例 強度 0.15〜0.25 の暗い青）＋各スポットライト。

## シーン（決定的・`Math.random` 不使用）
- 地面: 大判の平面/箱（**影を受ける** receiveShadow）。色は中明度のグレー。
- 柱: **8 × 8 = 64 本**のボックス。間隔 **6**、中心揃え。
  - 柱 i（c=i%8, r=floor(i/8)）: `x=(c-3.5)*6`、`z=(r-3.5)*6`。底面 y=0。
  - サイズ: 断面 **2 x 2**、高さ **h = 3 〜 9**（シード付き疑似乱数 mulberry32, seed=0x9e3779b9 で決定的）。中心 y = h/2。
  - 柱は**影を落とし、影を受ける**（castShadow + receiveShadow）。色は明るめグレー。

## 光源（スポットライト・影あり）
- **光源数 N**: 初期 **4**。`+`/`-`（`]`/`[`）で **±2**（**最小 1 / 最大 12**）。**比較の主軸**。
- 各光源 i は上空を周回（決定的・時間ベース）:
  - 高さ **y = 30**、周回半径 **22**、位相 `φ_i = i * 2π / N`、角速度 **0.4 rad/s**:
    `pos = (22 cos(t*0.4 + φ_i), 30, 22 sin(t*0.4 + φ_i))`。
  - 常に**シーン中心 (0, 1, 0) を指す**（target）。
  - 色: 色相を光源ごとに変える（`hue = i / N`、彩度高め・明るめ）。
  - スポット角 ≈ **50°**（penumbra/ソフトネスは各エンジンの近い値）、距離減衰で底まで届く強度。
  - **castShadows = true / シャドウマップ 1024**。
- N を変えたら光源群を作り直し、位相 φ_i を N に合わせて再配置。

## 操作
- `+` / `-`（`]` / `[`）: 影付き光源数の増減。**比較の主軸**。
- `R`: 光源数を初期値に戻す。
- カメラ固定・光源は自動周回（無人ベンチ可）。

## 共通HUD（必須）
画面左上にHTMLオーバーレイ:
- `FPS`（実測・移動平均）
- `Objects`（柱の数 = 64）
- `Lights`（現在の影付き光源数 N）
- `Draws` / `Tris`（各エンジンの統計。**シャドウパスを集計に含めるかはエンジン差がある**＝three.js の `renderer.info` はメインパスのみ計上しシャドウパスを含めない。Babylon/PlayCanvas の計測値とは意味がずれる点に注意）。
- **影コストの主指標は FPS**（光源数を増やすとシャドウマップ生成パスが増え FPS が落ちる）。

## アセット
- **画像/GLB不使用・プリミティブ（box / plane）のみ**。色は固定。
