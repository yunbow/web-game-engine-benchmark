# 3D テーマ9（T9）共通仕様 ― 3D ナビ群衆（グリッド A* 経路探索＋多数追従）

全ライブラリ（three.js / Babylon.js / PlayCanvas / A-Frame）で**同一仕様**を実装し、性能を横並び比較するための共通仕様。
**比較の主軸 = 多数エージェントの経路探索(A*)＋経路追従更新（CPU側AIロジック）と移動メッシュ描画**。2Dテーマ6（タワーディフェンス／A*）の3D版。

> この SPEC.md が唯一の正。グリッド・障害物・A*方式・エージェント数・ゴール挙動・カメラ・HUD を4ライブラリで揃える。
> **A* は4ライブラリとも自前実装（同一アルゴリズム）**＝経路探索ロジックの結果は一致させる（物理エンジン等は使わない）。

## ゲーム内容
- 障害物のあるグリッド地面を、多数のエージェントが**共通ゴールへ A* で経路探索**して移動。**ゴールが定期的に移動**すると**全エージェントが一斉に再計算**する。
- **目的は性能比較**: `+`/`-` でエージェント数を増減し、A* 計算＋経路追従＋移動メッシュ描画のコストで FPS を測る。

## 描画バックエンド
- **全ライブラリ WebGL2 で統一**。エージェントは**インスタンス描画**（移動ごとに per-instance 変換を更新）。

## 画面・カメラ
- 描画サイズ **960 x 540（16:9 固定）**。背景 `#0e131a`。**Y軸上向き**。地面 y=0。
- カメラ固定（斜め見下ろし）: 位置 **(0, 70, 60)** / 注視 **(0, 0, 0)** / fov 55° / near 0.5 / far 600。
- 環境光 + 平行光1灯。

## グリッド / 障害物（決定的・`Math.random` 不使用）
- グリッド **GW × GW = 40 × 40**、セルサイズ **CSZ = 2**（ワールド 80×80, 中心揃え）。
  - セル (gx,gz) のワールド中心: `x=(gx-(GW-1)/2)*CSZ`, `z=(gz-(GW-1)/2)*CSZ`。
- **障害物**: シード付き疑似乱数（mulberry32, seed=0x9e3779b9）で各セルを約 **18%** の確率で壁にする（`rnd()<0.18`）。
  - 壁セルは通行不可。壁は高さ **3** のボックスで描画（インスタンス）。
- 地面: 大判の平面（暗color）。

## エージェント（決定的）
- **エージェント数 N**: 初期 **150**。`+`/`-`（`]`/`[`）で **±50**（**最小 20 / 最大 1000**）。**比較の主軸**。
- 初期位置: 壁でない自由セルから決定的に選ぶ（PRNG 続き）。エージェントは現在セルを保持。
- 形状: 円錐（cone, 高さ ~1.6・半径 ~0.5）。インスタンス描画。進行方向へ向ける（任意）。移動速度 **6 u/s**。
- 各エージェントは現在の**経路（セル列）**を保持し、次のセル中心へ向かって移動。到達でインデックス前進。ゴール到達後は待機。

## ゴールと一斉再計算
- ゴール: 自由セル1つ。明るいマーカー（円柱）で表示。
- **GOAL_MS = 4000ms ごと**にゴールを次の自由セルへ移動（決定的な列／PRNG続き）。
- **ゴール移動時、全エージェントが現在セル→ゴールの A* 経路を再計算**（＝CPU負荷のバースト）。

## A*（全ライブラリ共通の自前実装）
- **4近傍（上下左右）、マンハッタン距離ヒューリスティック**、バイナリヒープ等で実装。壁セルは展開しない。
- 経路が無い場合はその場待機（再計算時に再挑戦）。
- 4ライブラリで同一のグリッド・同一アルゴリズム＝同じ経路になること（HUD で再計算回数を観測）。

## 操作
- `+` / `-`（`]` / `[`）: エージェント数の増減。**比較の主軸**。
- `R`: エージェント数を初期値に戻す（再配置）。
- 入力なしでゴールは自動移動（無人ベンチ可）。

## 共通HUD（必須）
画面左上にHTMLオーバーレイ:
- `FPS`（実測・移動平均）
- `Objects`（エージェント数 = N）
- `Agents`（現在のエージェント数設定 N）
- `Repaths`（A* 経路計算の累計回数。ゴール移動ごとに +N される）
- `Draws` / `Tris`（描画統計・ベストエフォート）。

## アセット
- **画像/GLB不使用・プリミティブ（cone/box/cylinder/plane）のみ**。色は固定。
