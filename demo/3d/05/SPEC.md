# 3D テーマ5（T2）共通仕様 ― 広域地形 + フラスタムカリング / LOD / 描画距離

全ライブラリ（three.js / Babylon.js / PlayCanvas / A-Frame）で**同一仕様**を実装し、性能を横並び比較するための共通仕様。
**比較の主軸 = 広域シーンの可視範囲のみ描画（フラスタムカリング）＋距離LOD＋描画距離**。2Dテーマ3（大マップ・カリング）の3D版。

> この SPEC.md が唯一の正。配置・距離・LOD閾値・カメラ・HUD を4ライブラリで揃える。

## ゲーム内容
- 広い地面に**大量の木（プロップ）をグリッド配置**し、カメラが上空を周回飛行。
- **目的は性能比較**: `+`/`-` で**描画距離**を増減し、可視範囲のみ描画（カリング）＋遠方LODでどれだけ軽く保てるかを測る。

## 描画バックエンド
- **全ライブラリ WebGL2 で統一**。各エンジンの**自動フラスタムカリング**（視錐台外メッシュを描かない）に乗せ、
  距離カリング（drawDist超は非表示）と LOD（遠方は低ポリ）は**アプリ側で実装**する。

## 画面・カメラ
- 描画サイズ **960 x 540（16:9 固定）**。背景は空色 `#8fb8e6`。**Y軸上向き**。地面 y=0。
- 環境光 + 平行光1灯（上方）。
- **カメラは自動周回飛行**（決定的・時間ベース、入力不要）:
  - 半径 **R=140** の円周を高さ **y=26** で周回（角速度 0.15 rad/s）: `cam = (R cosθ, 26, R sinθ)`、`θ = t*0.15`。
  - 注視点は円の中心側やや前方の地面: `look = (R*0.4*cosθ, 2, R*0.4*sinθ)`（＝進行方向斜め下を見る）。
  - fov 60° / near 0.5 / far 1200。

## ワールド配置（決定的・`Math.random` 不使用）
- 木を **GRID × GRID = 100 × 100 = 10000 本**、間隔 **SP = 8** でグリッド配置（中心揃え、ワールド約 792×792）。
  - 木 i（c=i%GRID, r=floor(i/GRID)）: `x=(c-(GRID-1)/2)*SP`、`z=(r-(GRID-1)/2)*SP`、`y=0`。
  - 各木の**高さ・向きの個体差**はシード付き疑似乱数（mulberry32, seed=0x9e3779b9）で決定的に（高さ係数 0.8〜1.4・Y回転）。
- 地面: 大判の平面（800×800 程度）暗緑。

## 木メッシュと LOD（2段）
- **LOD0（近距離・詳細）**: 幹（細い円柱, 6〜8分割）＋葉（円錐, 8分割）の2メッシュ。
- **LOD1（中距離・低ポリ）**: 単一の低ポリ円錐（4分割）など、明確に三角形数が少ない簡略形。
- **非表示（遠距離）**: drawDist 超は描画しない。

## カリング / LOD / 描画距離（毎フレーム・アプリ側）
- **描画距離 drawDist**: 初期 **120**。`+`/`-`（`]`/`[`）で **±40**（**最小 40 / 最大 360**）。**比較の主軸**。
- 各木とカメラの水平距離 d を計算し:
  - `d > drawDist` → 木を**非表示**（オブジェクトごと visible=false）。
  - `lodDist < d ≤ drawDist`（`lodDist = drawDist * 0.5`）→ **LOD1**（低ポリ）。
  - `d ≤ lodDist` → **LOD0**（詳細）。
- 表示中の木は、各エンジンの**自動フラスタムカリング**で視錐台外がさらに除外される（→ Draws は InRange より少なくなり、カリングの効きが観測できる）。

## 操作
- `+` / `-`（`]` / `[`）: 描画距離の増減。**比較の主軸**。
- `R`: 描画距離を初期値に戻す。
- カメラは自動飛行（無人ベンチ可）。

## 共通HUD（必須）
画面左上にHTMLオーバーレイ:
- `FPS`（実測・移動平均）
- `Objects`（描画距離内の木の本数 = InRange、距離カリング後）
- `DrawDist`（現在の描画距離設定）
- `Draws`（draw call 数。フラスタムカリング後の実描画。InRange より小さくなるのが正常）
- `Tris`（三角形数）

## アセット
- **画像/GLB不使用・プリミティブ（円柱/円錐/平面）のみ**。色は固定（幹=茶、葉=緑、地面=暗緑）。
