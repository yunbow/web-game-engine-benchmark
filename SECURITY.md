# Security Policy

This repository contains only static benchmark demos (plain HTML/JS loaded via
CDN) and documentation. It does not include any server-side code, backend
services, or distributed packages, so there is no traditional attack surface
to secure.

If you find a security concern in the demo code (for example, an unsafe
coding pattern shown in one of the examples), please report it via
[GitHub Issues](https://github.com/yunbow/web-game-engine-benchmark/issues).
